# ADR 0787: Redact cursor debug output

## Status

Accepted

## Context

`Cursor` stores opaque pagination tokens supplied by clients or generated by
storage/query planners. Derived `Debug` exposed the raw cursor token in
diagnostics.

## Decision

Implement custom `Debug` for `Cursor`. Preserve only the presence of a cursor
while redacting the token value.

## Consequences

Diagnostics still show that a cursor existed. Raw pagination cursor tokens no
longer appear through cursor debug formatting or through containers that
delegate to `Cursor` debug output.
